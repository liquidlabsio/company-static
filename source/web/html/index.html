Our Packages

Architecture Workshops (2-days)

Streaming architectures are different. The dataflow model needs to be constructed using sound principles that separate the core flow from the observability, instrumentation and control planes. This 2 days session covers: overall technology strategy, overall solution architecture, kafka architecture, technology selection, high-level data flow modelling, devops and automation (learn more)

Modern tooling & implementation (2-8 weeks)

Tooling of streaming platforms is different. We can be deployed to plan, adopt and implement technology strategy; including Cloud, Automation, DevOps, Build, application framework (Quarkus, Micronaut), distributed logging and CI/CD pipelines (learn more)

Data centrically designed with care (2-8 weeks)

Implementation and planning sessions for Event Storming, Domain driven design, data flow, operations and modelling. Additionally, tehnical design for data centric functions such as data-governance, data-lineage, data-classification, events-as-an-API and more. (learn more)

Our Services

Architectural workshops

Covering event-driven-design, Kafka, central nervous system, cloudevents and multi-cloud to build a blue-print

Coding
